Hỗ trợ nhập từ:
JSON (trạng thái TH) · Excel .xlsx (danh sách hạng mục)
CSV hoặc TSV từ bảng Excel

📌 Hướng dẫn nhập dữ liệu

  • File JSON (xuất từ app): nhập trạng thái đánh dấu, chi phí TT và tháng hiện tại
  • File Excel .xlsx: đọc các cột — cột đầu có thể là STT, sau đó station, system, task (hoặc nội dung), unit, qty, schedule, executor, cost_plan
  • File CSV/TSV: tương tự Excel, dùng dấu phẩy hoặc tab làm phân cách
  • Tên cột không phân biệt hoa thường, app sẽ tự nhận dạng
  • Sau khi xem trước, nhấn Nhập dữ liệu để áp dụng
